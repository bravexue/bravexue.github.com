网络地址结构

网络套接字描述符

使用如下接口创建一个套接字：

domain ：地址族AF_INET、AF_INET6、AF_LOCAL等。
type ：套接字类型SOCK_STREAM、SOCK_DGRAM、SOCK_RAW、SOCK_PACKET、SOCK_SEQPACKET等。
protocol ：协议IPPROTO_TCP、IPPTOTO_UDP、IPPROTO_SCTP、IPPROTO_TIPC等。
protocol为0表示domain和type下默认的协议。

字节序

不同计算机系统可能采用不同的大小端编码，网络传输规定使用大端字节序，因此需要转换。

地址格式

为了适配任何通信场景，定义了通用的地址struct sockaddr，套接字接口都是使用该结构作为参数。
调用者需要将特定场景的结构强转成通用的结构。
因特网的地址结构为struct sockaddr_in,其中主要包含了2部分：IP地址和进程端口，这样可以唯一标识通信的两端。
