PyQt5由浅入深(1)开始

Python GUI开发可用的第三方库众多，其中比较流行有大名鼎鼎的PyQt和wxPython，这两个都是跨平台开源库，拥有众多的用户。
本系列详细整理了PyQt功能实现，为初学者提供入门和深入指导。

PyQt5说明

PyQt其实就是在Qt之上封装并提供Python接口，Qt是跨平台C++图形用户界面应用程序开发框架，Qt的所有功能原理比如独特对信号槽机制都适用PyQt，PyQt资料不足可以参考Qt。PyQt有PyQt4和PyQt5两个版本，两者不兼容，建议与时俱进直接使用QyQt5。
本系列范例平台：macOS, Python3.6, PyQt5.9。
